快捷键

shift家族

  • 水平跟竖直移动： shift + 鼠标移动
  • 等比例放大和缩小：shift + 鼠标拖拽形状
  • 每次旋转15度：shift + 旋转

ctrl家族

  • 快速放大：ctrl + 鼠标滚轮
  • 移动复制：ctrl + 鼠标移动
  • 快速复制：ctrl + D
  • 新建页面：ctrl + M
  • 组合：ctrl + G
  • 恢复：ctrl + Y

ctrl + shift家族

  • 快速水平/竖直移动复制：ctrl + shift + 鼠标移动
  • 等比例中心放大：ctrl + shift + 鼠标拖拽形状
  • 复制格式/粘贴格式：ctrl + shift + C/V
  • 解除组合：ctrl + shift + G

文本框操作

  • 文本框快速放大和缩小：ctrl A+ -> ctrl + [/]
  • 文本框快速加粗斜体下划线：ctrl + A -> ctrl + B/I/U
  • 文本框快速居中/左/右对齐：ctrl + A -> ctrl + E/L/R

文字图片的快速插入

  • 快速在页面中插入文字：alt -> N -> X -> X
  • 快速在页面中插入图片：alt -> N -> P -> D
