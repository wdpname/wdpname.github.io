如何使用hexo&github搭建属于自己的博客

发表于2022-09-27|更新于2022-09-27
|阅读量:

前提条件

  • 已下载node，git

  • 已下载Hexo

npm install hexo-cli -g

步骤如下

  1. 注册一个github仓库，且仓库名必须为你的github注册名.github.io

  2. 生成ssh keys （非密钥）更加好

    • 在任意目录下打开git bash， 输入以下命令

      其中的qq邮箱就是github注册时用的邮箱

  3. 找到2步骤中执行命令后生成的.ssh文件夹

  4. 打开.pub文件并复制其中的内容

  5. 然后到自己的github主页中找到

  1. 然后将复制的ssh粘贴到这里，随便一个标题，然后点击add按钮

  1. 测试ssh是否绑定成功

    在打开的git bash命令界面中执行下面的命令

    ssh -T git@github.com

出现以上字样，说明设置ssh成功

  1. 在本地新建一个文件夹（本地博客仓库）

    • 在这个文件夹里打开git bash命令行窗口

    • 执行hexo init，然后就在这个文件夹中生成了很多文件

    • hexo s 生成本地静态页面, 然后就可以打开一个本地页面

      localhost:4000

npm install hexo-deployer-git —save
