论如何快速捡起使用pytorch的一些感悟

发表于2023-01-10|更新于2023-05-12
|阅读量:

视频讲解：https://www.bilibili.com/video/BV1nL4y1b7oT/?spm_id_from=333.337.search-card.all.click&vd_source=e7df7b103657feb95364df561d4a15b3

  1. 查看cuda版本 — nvadia-smi
  2. 下载相应版本的cuda

https://developer.nvidia.com/cuda-downloads

  1. 下载相应版本的cudnn

https://developer.nvidia.com/rdp/cudnn-download

  1. 安装pytorch

即可

然后将cudnn中的lib，include，bin文件夹覆盖到cuda文件夹下

然后配置上面三个文件夹的环境变量，外加一个libnvvp文件夹的环境变量，都放在系统变量下的path新建即可。

首先下载miniconda3， 默认python版本为3.10（因为本人电脑的显卡是30系列，3060），conda版本为22.11

在anaconda 控制台下新建一个环境，名为hhh —》 conda create -n hhh python=3.10

进入到这个hhh环境， —-》 conda activate hhh

安装pytorch 1.13.1（且cuda版本为11.7） —-》

import torch —》 torch.cuda.is_available() 如果为True 则表示可以使用GPU

在pycharm中新建一个项目，设置python解释器（python.exe），我们需要在新建的环境中找到这个python解释器，才能使用pytorch

30系列显卡不支持11.1以下的CUDA

30，40系显卡建议使用python 3.10环境，因为30系显卡需要cuda 11及以上，而py3.6不能支持cuda11以上。
即使你使用命令下载11.x版本，也会自动给你下载cpu版本。即使你下载了cuda 10.3版本，后期调用cuda会告诉你不支持计算能力7以上的卡（20系以上）

显卡驱动
cuda工具包 （同pytorch一键安装）

安装 cuda 11.7.1
安装torch 1.13.1

conda 能保证package能work
pip不能保证

在cmd中检查cuda是否安装成功：nvcc -V

安装cuda之后，再安装对应版本的torch

conda install pytorch torchvision torchaudio pytorch-cuda=11.7 -c pytorch -c nvidia
