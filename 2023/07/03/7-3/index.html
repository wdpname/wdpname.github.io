我的发现

  • pigGO卡掉，直接任务控制台结束任务

  • openAI Translator 不能用了，要花钱

  • conference 会议
  • make a new submission 提交新的内容或申请
  • IEEE：是一个学术组织，非常牛逼
  • ICCT：是IEEE主持的一个通信相关专业的国际会议
  • CSRF：是跨站请求伪造（Cross-Site Request Forgery）
  • 中间人攻击：有被动式，主动式，SSL/TLS中间人攻击

复习重点：

  • 完全二叉树
  • 最小堆
  • sql语句
  • 时间复杂度
  • 电梯调度算法
  • 多个线程并发
  • shell脚本
  • 快速排序
  • 线程和进程的区别
  • TCP 的粘包问题： 接收方无法分辨出传输层中每个数据包的边界
  • 分布式锁的实现方式：分布式锁即在分布式系统上实现锁

困惑

A中，当4出现在最左边，那么10就出现在左边第二个，在对10进行快速排序后，10的左边必定是6和8，那么第三次按照题意是12在4的第二个位置，那么就与题意相悖，所以A是不可能是第三次排序结果。
